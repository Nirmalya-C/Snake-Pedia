# Snake-Pedia 🐍

An interactive browser-based snake game where you compete against an AI opponent. Challenge yourself to collect more food than the AI and become the ultimate snake champion!

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![GitHub Pages](https://img.shields.io/badge/demo-live-brightgreen)](https://nirmalya-c.github.io/Snake-Pedia/)

## 🎮 Features

- **Player vs AI**: Compete against an intelligent AI opponent
- **Responsive Controls**: Use arrow keys or WASD for smooth gameplay
- **Real-time Scoring**: Track your score against the AI in real-time
- **Clean UI**: Modern, dark-themed interface with smooth animations
- **Boundary Wrapping**: Snakes wrap around edges for continuous gameplay

## 🚀 Quick Start

### Play Online

Visit the live demo: [https://nirmalya-c.github.io/Snake-Pedia/](https://nirmalya-c.github.io/Snake-Pedia/)

### Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Nirmalya-C/Snake-Pedia.git
   cd Snake-Pedia
   ```

2. Open `index.html` in your web browser:
   ```bash
   # On macOS
   open index.html

   # On Linux
   xdg-open index.html

   # On Windows
   start index.html
   ```

   Or use a local server:
   ```bash
   # Python 3
   python -m http.server 8000

   # Then visit http://localhost:8000
   ```

## 🎯 How to Play

1. **Objective**: Collect more food than the AI snake to win
2. **Controls**:
   - Use **Arrow Keys** (↑ ↓ ← →) or **WASD** to move your snake
   - Your snake is displayed in **green**
   - The AI snake is displayed in **orange**
3. **Scoring**: Each food item collected increases your score by 1
4. **Strategy**: Plan your moves carefully to reach food before the AI does!
